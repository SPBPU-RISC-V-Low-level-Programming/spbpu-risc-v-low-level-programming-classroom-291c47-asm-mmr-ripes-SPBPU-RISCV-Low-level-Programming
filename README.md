# SPBPU-RISCV-Low-level-Programming
Главный репозиторий, используемый для работы с лабораторными работами. Сюда будут подгружены все лабы, выполненные студентами и прошедшие ревью.

Правила выполнения лабораторных работ:
1) Лабораторные работы проходят code review через github classroom;
2) Сначала студенту необходимо:
  а) создать свою директорию в папке его группы с именем в виде фамилия_ио (например, моя папка выглядела бы skrebenkov_di);
  б) для каждой лабы создать директорию в папке со своим именем с указанием номера (lab_X, где X - номер лабы);
3) При загрузке выполненной работы в директории с лабой должны быть:
  а) исходные файлы;
  б) инструкция, написанная в файле README.md, в котором написано, как нужно собрать проект и запустить его.
    Например, для первой лабы в Ripes в README.md нужно написать, с какой скоростью запускать симуляцию, когда прожимать кнопки и т.д.
